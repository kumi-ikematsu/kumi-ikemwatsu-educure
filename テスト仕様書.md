# テスト仕様書

## プロジェクト名
英単語暗記アプリケーション（VocabularyApp）

## バージョン
1.0.0

## 作成日
2025-06-11

---

## 1. テスト対象クラスと概要

| クラス名 | 役割 |
|:---------|:-----|
| Word | 英単語・日本語訳のデータ保持 |
| DBManager | PostgreSQL接続・テーブル初期化 |
| WordManager | 単語のCRUD操作（addWord / getWords / getWordCount / deleteWord / updateWord） |
| Quiz | クイズ出題・採点（getRandomWord / checkAnswer） |
| FileHandler | CSVエクスポート / CSVインポート |
| VocabularyApp | メインメニュー・各機能の統合 |

## 9. 手動テスト手順（統合テスト）

以下の手順でアプリを起動し、コンソール出力を確認する。

### 9.1 単語登録 → クイズ → 削除の一連フロー

```
1. アプリ起動
2. メニューで「1」→ 英単語「apple」、日本語「りんご」を登録
3. メニューで「2」→ クイズ開始。「りんご」と入力 → 「正解です！」を確認
4. メニューで「5」→ 「apple」を入力 → 「単語を削除しました。」を確認
5. メニューで「2」→ 「登録された単語がありません。」を確認
6. メニューで「7」→ 終了
```

### 9.2 CSVインポート → エクスポートフロー

```
1. words.csv を用意（例: apple,りんご / banana,バナナ）
2. メニューで「3」→ ファイル名「words.csv」→ 「2個の単語を読み込みました。」を確認
3. メニューで「4」→ ファイル名「export.csv」→ 「2個の単語を保存しました。」を確認
4. export.csv の内容を目視確認
```

### 9.3 異常系：存在しない単語の更新・削除

```
1. メニューで「6」→ 「notexist」→ 「エラー: 指定した単語が見つかりません」を確認
2. メニューで「5」→ 「notexist」→ 「エラー: 指定した単語が見つかりません」を確認
```

## 10. 合格基準

- 全正常系テストで例外なし、期待出力が得られること
- 全異常系テストで適切な例外またはエラーメッセージが出力されること
- 性能テストで各操作が1秒以内に完了すること
- 手動統合テストで一連のフローが正常に完結すること
